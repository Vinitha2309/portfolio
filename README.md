## ⚙️ How to Run Locally

1. Download or clone the repo:

   ```bash
   git clone https://github.com/vinitha-dev/vinitha-dev.github.io.git
   ```

2. Open folder:

   ```bash
   cd vinitha-dev.github.io
   ```

3. Run:

   * Open `index.html` in browser

---

## 🌍 Deployment (GitHub Pages)

1. Create repo:

   ```
   vinitha-dev.github.io
   ```

2. Upload files

3. Go to:

   ```
   Settings → Pages → main branch → Save
   ```

4. Your site will be live 🎉
